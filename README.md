# Elif Birthday Hub 🌙

Un site d'anniversaire personnalisé avec fond de tapis turc 70s kitsch pour ma chérie.

## 🎮 Le Système de Layers (Niveaux)

Le hub évolue maintenant sous forme de "Layers" débloquables grâce au score global !

### Layer 1 (Classiques)
- **👾 Space Invaders** — Défends la galaxie
- **🀄 Mahjong Solitaire** — Trouvez les paires de stickers
- **🔢 Sudoku** — Remplis la grille avec des mots doux
- **🧱 Tetris** — Empilement classique
- **🪓 Uzinagaz (Lancer de hache)** — Vise bien !

### Layer 2 (Jeux 2D Avancés - *En développement*)
- **🍄 Super Elif** — Jeu de plateforme (style Mario)
- **🐦 Angry Stickers** — Lance-pierre physique
- **🏰 Tower Defense** — Protège la base

### Layer 3 (Expériences 3D - *En développement*)
- **🔫 Counter Stickers** — Et bien d'autres expériences en 3D !

## ✨ Fonctionnalités

- Overlay d'entrée kitsch avec texte clignotant
- Message personnel personnalisable
- Fond "collage dynamique" dense et animé à partir de stickers
- **Score global persistant** — chaque victoire augmente ton score.
- **Système de sauvegarde** — Génère un code pour reprendre ta partie sur tablette !
- **Porte Magique** (Layer 2) — Débloquée à 6000 points (Nouveaux jeux + Surprise vidéo).
- **Porte Magique 3D** (Layer 3) — Débloquée à 12000 points.
